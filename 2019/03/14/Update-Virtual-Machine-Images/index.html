Update: Virtual Machine Images

2019-03-14
Virtual Machine, motd

Update to the Virtual Machine Images, the Lab machines have had a minor update, the motd (Message of the day) has been changed to reflect context regarding the 2 most important issues that trouble users of the course, a) The keyboard configuration not working whilst using the VM console and b) DNS not working, when the dnsmasq VM is not running.

The MOTD looks like the following -

Within the MOTD, on Ubuntu I’ve also disabled the advice checker that tells you that you can upgrade the virtual machine to the latest version of Ubuntu.

I believe some users have seen this and have thought of it as a requirement prior to continuing. If you wish to, you can update the lab environment to the latest version of Ubuntu, however it is not a requirement and some of the playbooks that template the likes of nginx configuration may not work, if there’s been an upstream change to the base configuration.

Given that the lab has been designed to match the context of the course, I advise that you don’t upgrade the environment, unless you’re happy to address subsequent changes that may arise to the course content.
