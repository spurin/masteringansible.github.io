Virtual Machine Download

2019-03-10

The virtual machines used for the lab in the course, can be downloaded using the following shortened url -

https://goo.gl/b68auU

I’ve been contacted in the past and advised that this url has failed individuals, upon checking I’ve not seen any issues but thats not to say there wasn’t an intermittent failure. Google have discontinued their url shortening service but have advised that they will continue the url shortening service for existing urls. Should you face an issue, the full url for the lab hosts is as follows -

https://drive.google.com/drive/folders/1QBgNs6knbwpSPyUMexDXhKjv3TO_Ka55
